If the VRAM bank is 1 and the new value has bit 7 set but the old value didn\textquotesingle{}t, it increments the number of tiles that use tile priority for that map. If the new value doesn\textquotesingle{}t have bit 7 set but the old value did, it decrements the number of tiles that use tile priority for that map.
